# Cannon-Manufacture-Usage-Guide-
本项目系统整理迫击炮、高射炮及自主设计火炮的制作流程、图纸、火药制备、数学计算与操作规范，基于理论研究。This project systematically compiles the manufacturing processes, drawings, gunpowder preparation, mathematical calculations, and operational specifications for mortars, anti-aircraft guns, and self-designed cannons, based on theoretical research.
